1. Introduction

1.1 Welcome to Appeon PowerBuilder 2021

Welcome to Appeon PowerBuilder!

Appeon PowerBuilder is an enterprise development tool that allows you to build data-driven business applications and components. It is one of a suite of Appeon products that together provide the tools to develop client/server or cloud applications.

1.2 Components of PowerBuilder

PowerBuilder 2021 contains the following products/components.

  • PowerBuilder IDE - A PowerScript development platform for rapidly developing cloud or client/server applications for Windows devices.
  • PowerBuilder Runtime - Runtime files for applications developed with the PowerBuilder IDE.
  • PowerBuilder Compiler - A standalone tool for compiling PowerBuilder projects using Windows DOS commands.
  • PowerServer Toolkit - A plug-in for the PowerBuilder IDE that compiles projects as installable cloud apps with REST APIs for the PowerServer runtime on top of the cross-OS .NET Core framework.
  • SnapDevelop - A lightweight yet powerful C# IDE for rapidly developing web or cloud applications on top of the cross-OS .NET Core framework.
    • DataWindow Converter - A SnapDevelop plug-in that automatically generates .NET DataStore from existing PowerBuilder DataWindows.
    • PowerScript Migrator - A SnapDevelop plug-in that automatically translates non-visual PowerBuilder code and objects (including PowerScript, embedded SQLs, DataWindows/DataStores, and NVOs) to C#.

2. System Requirements

Before installing PowerBuilder 2021, please ensure that your system meets the requirements below:

For installing PowerBuilder IDE & PowerServer Toolkit

  • Windows 10 or 8.1
  • (Optional) Microsoft .NET Framework 4.6.1 SDK or later (for using the .NET DLL Importer in the PowerBuilder IDE) (will be automatically installed by the setup program)
  • (Optional) SAP SQL Anywhere 17.0 or 16.0, or PostgreSQL 13, 12, or 11.3 (for running the PowerBuilder demos)

For installing SnapDevelop

  • Windows 10 or 8.1
  • Universal C Runtime
    Note: The setup program will check if Universal C Runtime is installed; if it is not installed, the setup program will provide the package list, and you will need to manually download and install the package.
  • Microsoft .NET Framework 4.8 SDK, Microsoft .NET Core SDK 2.1.500 and 3.1.302, & TypeScript SDK 3.1 (will be automatically installed by the setup program)

3. Installing PowerBuilder 2021

The PowerBuilder installation package can install the PowerBuilder IDE, PowerBuilder Runtime, PowerBuilder Compiler, PowerServer Toolkit, and SnapDevelop.

Step 1: Log into the Appeon User Center, and download the PowerBuilder Installer executable from the Downloads page on the Appeon User Center.

The PowerBuilder Installer executable (PowerBuilderInstaller_bootstrapper.exe) is available in your download directory after it is downloaded.

Step 2: Double click PowerBuilderInstaller_bootstrapper.exe to start the PowerBuilder Installer (you must have administrator privileges to run the installer).

Step 3: Select to agree to the license terms and click Continue.

Step 4: Click Install under PowerBuilder 2021.

Step 5: Select PowerBuilder IDE, PowerBuilder Compiler, PowerServer Toolkit, and SnapDevelop from the Programs tab; and then select PowerBuilder Runtime from the Components tab.

Step 6: Click Install to begin the installation.

Step 7: When the installations are completed, click Restart now to reboot the system.

4. Activating PowerBuilder 2021

You are required to activate PowerBuilder IDE, SnapDevelop, and PowerServer before using it. If you do not have a valid account, please contact your license administrator to create an account for you. And depending on whether the machine where PowerBuilder IDE and/or SnapDevelop is installed has Internet access or not, the account should be assigned with an online license or offline license by your license administrator.

You will need to have a PowerBuilder CloudPro license in order to use the PowerServer deployment features in the PowerServer IDE.

Follow the "Activating PowerBuilder" section in the Appeon License User Guide to activate PowerBuilder IDE; follow the "Activating SnapDevelop" section to activate SnapDevelop; and follow the "Activating PowerServer 2021" section to activate PowerServer.

5. Uninstalling PowerBuilder 2021

5.1 Uninstalling PowerBuilder

Before you uninstall PowerBuilder, make sure you have logged out PowerBuilder IDE if using an online license, or have deactivated the offline license. For how to log out PowerBuilder IDE or deactivate an offline license, refer to Logging out or De-activating an offline license in the Appeon License User Guide.

Launch Uninstall a program in the Windows Control Panel. Select PowerBuilder IDE 2021, PowerBuilder Compiler 2021, and PowerBuilder Runtime 2021, and click the Uninstall button to begin the uninstall process.

5.1 Uninstalling PowerServer

Launch Uninstall a program in the Windows Control Panel. Select PowerServer Toolkit 2021 and click the Uninstall button to begin the uninstall process.

5.3 Uninstalling SnapDevelop

Launch Uninstall a program in the Windows Control Panel. Select SnapDevelop 2021 (PowerBuilder) and click the Uninstall button to begin the uninstall process.
