Release notes

The Appeon PowerBuilder 2021 Maintenance Release 1311 is a setup program that updates the following products/components installed from the installation package of Appeon PowerBuilder 2021 Build 1288 (English) or later.

  • PowerBuilder IDE
  • PowerBuilder Runtime
  • PowerBuilder Compiler
  • PowerServer Toolkit
  • SnapDevelop (and plug-ins: PowerScript Migrator and .NET DataStore)

For detailed release notes (including bug fixes and improvements) of this maintenance release, refer to https://docs.appeon.com/pb2021/release_bulletin_for_pb/Build-1311-MR.html.

Checking product versions

This Maintenance Release (hereinafter referred to as MR) can only be applied to Appeon PowerBuilder 2021 Build 1288 or later. The MR setup program will automatically check which products and versions are already installed before performing any update. You can also check the product and version with the following steps:

Step 1: Open Windows Control Panel\Programs\Programs and Features.

Step 2: In the list of currently installed programs, select the product (for example "PowerBuilder IDE 2021"), and view the product version at the bottom of the window. For example, 21.0.0.1288 indicates 2021 Build 1288.

After MR installation, you can check the product versions using the same instructions, the product version should read 21.0.0.1311.

MR installation note

Run the MR setup program on the computer where a product needs updating.

MR silent installation

You can also run the MR setup program from a command line in the silent mode:

Step 1: Download the offline installation package for MR to the local disk of the computer.

Step 2: Open a Windows command window, and change to the directory of the MR installation package on the local computer.

Step 3: Execute the silent installer: SilentInstall.bat.

<<TOP

MR uninstallation note

There is no separate uninstall program for MR. Uninstalling the product will automatically uninstall the MR.
